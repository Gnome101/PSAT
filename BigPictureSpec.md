# Submitting New Protocols

### Filling out information

When I submit a new protocol, I can provide any amount of information

- Company Name\*
- Company Website
- Important/Known Addresses
- Github Page
- Docs Page

The company name is the only required one.

### Initial Address Discovery

This process runs at the start of the pipeline and consits of an AI search and a defilama search.


### Continuous Address Discovery

- Daap Crawler

Crawlers from U.S,Europe, and Asia can crawl the DaAPP searching for new contracts to interact with.

- Important Address Monitoring

EOAs, Safe's, and Timelocks tied to Ether Fi contracts can be monitored for additional deployments.
