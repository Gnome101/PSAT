# Submitting New Protocols

### Filling out information

When I submit a new protocol, I can provide any amount of information

- Company Name\*
- Company Website
- Important/Known Addresses
- Github Page
- Docs Page

The company name is the only required one.

### Address Discovery

This process runs at the start of the pipeline and consits of an AI search, a defilama crawl, and a daap crawl.
